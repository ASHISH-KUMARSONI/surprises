# Surprises & Mini-Games Repository 💖

A collection of cute, interactive surprises, mini-games, and lovely pranks for your crush. Built as a lightweight static site for GitHub Pages.

## 🌈 Project Philosophy
- **Extensible**: Each surprise is a standalone folder. Add as many as you want!
- **Non-Guessable**: Surprises are stored in folders with random hashes (e.g., `s/be-mine-2b7k/`) to keep them private until shared.
- **Creativity-First**: No need for design consistency. Each page is a unique experience.

## 🛠️ How to Add a New Surprise
1. Create a new folder inside `/s/` with a non-guessable name: `mkdir s/love-letter-x9y2`
2. Add your `index.html`, `style.css`, and `main.js`.
3. (Optional) Put shared images/fonts in `/assets/`.
4. Deploy by pushing to GitHub!

## ✨ Features
- Handcrafted CSS-only characters.
- Lightweight animations.
- Mobile-friendly interactions.
- Celebrations with `canvas-confetti`.
